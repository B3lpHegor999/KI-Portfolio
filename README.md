# KI-Portfolio: Persona-Logik & Memory-Architektur

⚠️ **Hinweis:** Dieses Repository befindet sich aktuell im aktiven Umbau und wird kontinuierlich mit Projektdaten sowie Dokumentationen bestückt.

## 🚀 Kernprojekte

### 1. Shift-Master AI (Persona-Logic)
Entwicklung einer spezialisierten Logistik-Persona für die Personalplanung. 
- **Fokus:** Mathematisch valide Schichtpläne ohne menschliche Fehlerquote oder KI-Halluzinationen.
- **Validierung:** Integration von harten Compliance-Regeln (z. B. 11h Ruhezeit), die direkt über den Python Code Interpreter geprüft werden.

### 2. Gemini Memory Exporter (Tooling)
Ein Browser-Utility (Tampermonkey), um Chat-Verläufe strukturiert für die Weiterverarbeitung in KI-Gedächtnis-Systemen aufzubereiten.
- **Funktion:** Automatisches Tagging von Rollen (User/Chatbot) für nahtloses Chunks-Management.

## 🛠️ Tech-Stack
- **Sprachen:** Python, PowerShell, JavaScript
- **Konzepte:** Prompt-Engineering, Identity-Seeding, AI-Human-Teaming
